# Projet Peinture et Rénovation

Ce projet est un site web vitrine développé pour une entreprise de peinture et rénovation. L'objectif est de présenter les services de l'entreprise, ses réalisations, ainsi que de faciliter la prise de contact avec les clients potentiels.

## Technologies Utilisées

- **HTML5** : Structure du site
- **CSS3** : Mise en forme et responsive design
- **JavaScript** : Interactions dynamiques
- **Vue.js** : Composants dynamiques et gestion de l'interactivité (si applicable)
- **PHP** : Gestion des formulaires de contact (si applicable)
- **MySQL** : Base de données pour stocker les demandes de contact (si applicable)

## Fonctionnalités

- **Page d'accueil** : Présentation de l'entreprise et de ses services
- **Galerie de réalisations** : Affichage des travaux effectués
- **Témoignages clients** : Avis des clients pour renforcer la crédibilité
- **Formulaire de contact** : Permet aux utilisateurs de faire des demandes de devis
- **Design Responsive** : Adapté aux mobiles, tablettes et ordinateurs
- **SEO Friendly** : Optimisation pour le référencement naturel

## Installation et Exécution

1. **Cloner le dépôt**
   ```sh
   git clone https://github.com/Amadou39/Projet-Peinture-R-novation.git
   ```
2. **Ouvrir le projet**
   - Si le projet utilise uniquement HTML/CSS/JS, ouvrir `index.html` dans un navigateur.
   - Si un serveur est requis (PHP/MySQL), utiliser XAMPP/WAMP et placer les fichiers dans le dossier `htdocs`.
   - Pour Vue.js, s'assurer d'avoir Node.js installé puis exécuter :
     ```sh
     npm install
     npm run dev
     ```
